# Kan Documentation

Kan is a file-based kanban board. All data lives as plain files in `.kan/` — no database, no server dependencies, works with any VCS.

**A note on shortcuts:** Throughout these docs, ⌘ represents the super key — Cmd on Mac, Ctrl on Windows/Linux.
